22. Pourquoi doit-on stationnariser les séries ?¶

**On doit stationnariser les sries pour sassurer de pouvoir les modliser laide de la mthode des MCO. En effet, pour une srie non-stationnaire, lhypothse dindpendance des observations nest plus valable. Le thorme centrale limite ne sapplique plus et lestimateur des MCO nest plus sans biais.
D'autre part, la distribution des paramtres de la regression ne suit plus une loi de Student.**

def get_p_df(df_ts):
    print '\nAR(p) for time series : ', str(ts)
    monthly_timeseries = pd.Series(df_ts.tolist(), index=monthly_timeserie_date, dtype=float)

    #Here we have to use the monthly timeserie as the statsmodels ar_model doesn't understand hourly frequencies
    ar_mod = sm.tsa.ar_model.AR(monthly_timeseries)
    
    #Selection of the optimal order for AR model based on bic
    res_order_aic = []
    res_order_bic = []
    maxlag = 20
    for i in range(0, maxlag):
        ar_res = ar_mod.fit(i)
        res_order_aic.append(ar_res.aic)
        res_order_bic.append(ar_res.bic)

    # plot
    fig, (ax1, ax2) = plt.subplots(1, 2)
    fig.set_size_inches(16, 4)
    pd.DataFrame(res_order_aic, columns=['aic']).plot(ax=ax1)
    legend = ax1.legend(loc = 'upper right')
    pd.DataFrame(res_order_bic, columns=['bic']).plot(ax=ax2)
    legend = ax2.legend(loc = 'upper right')
    plt.show()
    
    # get the minimum p
    min_p_aic = res_order_aic.index(min(res_order_aic))
    min_p_bic = res_order_bic.index(min(res_order_bic))
    p =  min([min_p_aic, min_p_bic])
    print 'minimum p for aic : ', str(min_p_aic)
    print 'minimum p for bic : ', str(min_p_bic)
    print 'selected p        : ', str(p), ' with ', 'aic' if min_p_aic < min_p_bic else 'bic'
    return ar_mod, p

p_df = dict()
p_df['pcsp'] = get_p_df(df['pcsp'])
p_df['divyld'] = get_p_df(df['divyld'])

Rappel sur le test non paramtrique de Durbin-Watson :

  • Sa valeur est comprise entre 0 et 4.
  • Une valeur proche de zro indique une autocorrlation positive
  • les valeurs situes autour de 2 montrent une absence dautocorrlation
  • si lon sapproche de 4, il existe une autocorrlation ngative (valeurs tantt au-dessus et tantt au-dessous de la tendance).

**Soit l'hypothse H0 disant qu'il n'y a pas de correlations des erreurs et H1 : il existe un processus autorgressif dordre 1.
On cherche savoir si partir de notre valeur retourn par notre test de Durbin-Watson, on accepte ou non l'hypotse nulle.
Pour ce faire, nous regardons les tables de Durbin-Watson (http://web.stanford.edu/~clint/bench/dw05d.htm) 5%, n=550 (taille de l'chantillon), et k=2 (nombre de variables explicatives, intercept inclus).
Cette table nous donne deux limites entre lesquelles il y a un doute (dL et dU).
Le DW (1.521) se situant entre 0 et 1.85632, on rejette H0. Il se peut quil existe une autocorrlation positive dordre 1.**
